# PDAI Calculator

Online calculator for the Pemphigus Disease Area Index (PDAI) — a validated scoring system for assessing pemphigus disease severity.

## Features

- Scoring for skin (12 areas), scalp, and mucosa
- Automatic severity classification (mild / moderate / severe)
- Body and oral cavity visualization
- PDF and print export
- Russian, English, Uzbek, and Karakalpak interface

## Stack

Vite + React, Tailwind CSS, react-i18next

## Development

```bash
npm install
npm run dev
```
